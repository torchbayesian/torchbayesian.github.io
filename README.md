# torchbayesian.github.io

Deployment repository for the GitHub page.

##### Instructions for deployment

1. Clean and build page in `torchbayesian` repo;
2. Copy everything in `docs/build/html/` of `torchbayesian` repo;
3. Paste in emptied `docs/` of `torchbayesian.github.io` repo;
4. Create a file named `.nojekyll` in `docs/` of `torchbayesian.github.io` repo.
